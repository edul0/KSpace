# #  KSpace | Collaborative Kanban System

![KSpace Logo](marca.png)

**KSpace** é uma plataforma de gestão visual de tarefas (Kanban) desenvolvida para oferecer colaboração em tempo real com alta performance e design minimalista. Focado em produtividade ágil, o sistema permite que equipes gerenciem fluxos de trabalho de forma intuitiva, tanto em desktop quanto em dispositivos móveis.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando a "Trindade do Front-end" aliada a um Backend-as-a-Service (BaaS) de elite:

* **HTML5 & CSS3:** Arquitetura de layout responsiva com variáveis dinâmicas e design baseado em Post-its.
* **JavaScript (ES6+):** Lógica de manipulação de DOM, eventos de Drag & Drop e gerenciamento de estado.
* **[Supabase](https://supabase.com/):** * **PostgreSQL:** Armazenamento persistente dos murais e logs.
    * **Realtime Engine:** Sincronização instantânea entre múltiplos dispositivos via WebSockets.
    * **Presence:** Monitoramento de usuários online na sala em tempo real.

---

## ✨ Funcionalidades de Elite

* **⚡ Realtime Sync:** Qualquer alteração (criar, mover ou deletar) é refletida instantaneamente para todos os usuários na sala.
* **👥 Presence System:** Saiba exatamente quem está online no mural através da aba lateral de presença.
* **🖱️ Intuitive Drag & Drop:** Movimentação fluida de tarefas entre as colunas "A Fazer", "Em Curso" e "Concluído".
* **📌 Gestão de Prioridades:** Sistema de sinalização por emojis (📌 Alta, ⚡ Média, 🍃 Baixa) com feedback visual por cores.
* **🖼️ Anexos de Mídia:** Suporte para inclusão de imagens diretamente nos Post-its.
* **👤 Atribuição de Tarefas:** Delegue responsabilidades vinculando usuários (via GitHub ou Manual) a cada card.
* **🌓 Adaptive Theme:** Suporte total a Dark Mode para conforto visual em longas jornadas.
* **☢️ Nuke Command:** Função administrativa para resetar o mural e limpar o fluxo de trabalho.
* **📜 Histórico de Atividade:** Log persistente que registra cada ação realizada na sala.
